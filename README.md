# Лабораторная работа №3 — Криптосистема Эль-Гамаля (Вариант 2)

## Описание

Реализация алгоритма шифрования Эль-Гамаля для файлов произвольного формата на C# (.NET 8, Avalonia UI — кроссплатформенный GUI для macOS/Windows/Linux).

### Что реализовано

| Требование | Реализация |
|---|---|
| Шифратор/дешифратор Эль-Гамаля | `ElGamalCipher.cs` |
| Алгоритм быстрого возведения в степень | `ElGamalCipher.FastExp()` |
| Нахождение **всех** первообразных корней mod p | `ElGamalCipher.FindAllPrimitiveRoots()` |
| Шифрование файла произвольного формата | `EncryptBytes()` — побайтово |
| Вывод зашифрованного файла числами (10-я система) | Область "Зашифрованный файл" в GUI |
| Проверка ограничений на параметры | Валидация p (простое), g (первообразный корень), x, k (взаимно простое с p−1) |
| Вычисление открытого ключа y = g^x mod p | Отображается в реальном времени |
| Результат — зашифрованный/расшифрованный файл | Сохраняется на диск |

## Описание алгоритма

### Генерация ключей

1. Задаётся простое число **p**
2. Выбирается **g** — первообразный корень по модулю p  
   (g является первообразным корнем ⟺ g^((p−1)/q) ≢ 1 (mod p) для всех простых делителей q числа p−1)
3. Задаётся **x** — закрытый ключ: 1 < x < p−1
4. Вычисляется **y = g^x mod p** — часть открытого ключа

**Открытый ключ Ko = (p, g, y)**  
**Закрытый ключ Kc = (x, p)**

### Шифрование

Для каждого байта m файла (0 ≤ m < p):
- Задаётся сеансовый ключ **k**: 1 < k < p−1, НОД(k, p−1) = 1
- `a = g^k mod p`
- `b = y^k · m mod p`
- Результат: пара **(a, b)** — записывается в выходной файл

### Дешифрование

Для каждой пары (a, b) шифротекста:
- `m = b · a^(−x) mod p = b · a^(p−1−x) mod p`  
  (по малой теореме Ферма: a^(p−1) ≡ 1 (mod p))

### Быстрое возведение в степень

```
function fast_exp(a, z, n):
    a1 := a mod n; z1 := z; x := 1
    while z1 ≠ 0:
        while z1 mod 2 = 0:
            z1 := z1 / 2
            a1 := (a1 * a1) mod n
        z1 := z1 − 1
        x := (x * a1) mod n
    return x
```

## Параметры для варианта p = 41

При p = 41:
- φ(41) = 40 = 2³ · 5
- Первообразных корней: φ(40) = **16 штук**
- Корни: 2, 6, 7, 11, 12, 13, 17, 22, 24, 26, 28, 29, 30, 34, 35, 37

Пример рабочих параметров:
```
p = 41
g = 7   (первообразный корень mod 41)
x = 15  (закрытый ключ)
k = 11  (НОД(11, 40) = 1 ✓)
y = 7^15 mod 41 = 38  (открытый ключ)
```

> ⚠️ При p = 41 можно шифровать байты **только от 0 до 40** (m < p).  
> Для шифрования произвольных файлов (байты 0–255) используйте p > 255, например p = 257.

## Формат зашифрованного файла

Текстовый файл, каждая строка — пара чисел в десятичной системе:

```
a₁ b₁
a₂ b₂
...
aₙ bₙ
```

где n — количество байт исходного файла.

## Запуск тестов

Тест-файл `ElGamalTests.cs` — автономный, не требует xUnit.  
Создайте отдельный консольный проект или добавьте в существующий:

```bash
cd ElGamalLab
dotnet new console -n Tests
cp ElGamalTests.cs Tests/
# Скопируйте туда ElGamalCipher.cs и запустите:
cd Tests && dotnet run
```

Тесты проверяют:
- Быстрое возведение в степень (пример из учебника: 7^5 mod 13 = 11)
- Проверку простоты
- Разложение на простые множители
- Нахождение всех первообразных корней
- Полный пример из учебника (стр. 18): p=13, g=7, x=5, k=7, m=4 → (a=6, b=8)
- Шифрование/дешифрование всех байтов при p=41
